Step 4 — Dynamic Routes (Very Important):
------------------------------------------

@app.route("/user/<name>")
def user(name):
    return "Hello" + name 

now open:
http://127.0.0.1:5000/user/john

output:
Hello john


step 5 Returning JSON(Real APIs do this )
----------------------------------------

from flask import Flask,jsonify

app = Flask(__name__)

@app.route('/weather')
def weather():
    data = {
        "temperature":23,
        "humidity":59
    }
    return jsonify(data)
app.run()

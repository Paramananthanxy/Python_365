Cursor:
-------------

cursor.execute() --> for single sql query.. select,insert,create
cursor.executemany() --> run the same query multiple times 
data = [
("2024-01-01",15,36,17.0),
("2024-01-02",31,28,15.1)
]

cursor.executemany(
"INSERT INTO weather VALUES (?, ?, ?, ?)",
data
)

cursor.fetchone()  ---> for fetching one rows
cursor.fetchall()  ---> for fetching many rows
cursor.fetchmany() ---> for fetching few rows at a time 
    rows = cursor.fetchmany(2)
cursor.close()     ---> for closing the connection 
cursor.rowcount    ---> telling how many rows where affected 

    cursor.execute("DELETE FROM weather WHERE date='2024-01-01'")
    print(cursor.rowcount)
